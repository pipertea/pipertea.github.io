Code Log 4: CSS Color

What do CSS properties do?

  1. Color: helps establish webpage elements regarding text, background, and border in contrast, visual hierarchy, and readability
  2. Background-Color: controls the background elements of the specified color including the webpage's content, padding, and border
  3. Border-Color: controls the color of the border when specifically written in code

How does CSS accept color as a value? Provide examples of colors in these styles:

CSS can accept colors as a value through many different predefined modern vocabulary and numerical systems.

  1. Named: red, aqua, olive
  2. Hexadecimal: #FF0000 for red, #800080 for purple, #ADD8E6 for light blue
  3. RGB: rgb(255, 0, 0) red, rgb(0, 255, 0) green, rgb(0, 0, 255) blue
