CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media uery works by starting off with a media and media type tag rule specifying how it is looked at based on a user's browsing device, conforming to whatever format the code insists. Characteristics include viewport/page characteristics, display quality, color, interaction, video prefixed, scripting, and user preference. The use of media queries allows for all the users to engage with material appearing on the web despite their difference in device formats or abilities.

What "media features" are supported by media queries--such as "orientation" or "max-width"--and what do they do?

  1. aspect-ratio: defines the width-to-height aspect ratio of the viewport
  2. resolution: defines the target pixel density of the device
  3. grid: determines if the device uses a grid (1) or bitmap (0) screen
  4. monochrome: the number of bits per pixel that a device's monochrome supports, where zero is no monochrome support.
